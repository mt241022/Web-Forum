# Web-Forum
A web forum where users can create accounts and, once logged in, start new threads and post replies under existing threads. This project served as an introduction to PHP and working with MySQL databases.

## Technologies
- PHP
- MySQL
- HTML & CSS

## Setup
This project requires a MySQL database. Database credentials are not included for security reasons. 
To run the project, import database.sql to create the required tables and configure your own credentials in the config.inc.php file.
